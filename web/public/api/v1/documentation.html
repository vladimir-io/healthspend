Developer API

Automated price discovery, market context aggregation, and programmatic enforcement triggers for your medical bill negotiation platforms.

GET /API/V1/SEARCH
Audit Ledger Search

Execute deep-audits against the global pricing index. Supports semantic procedure queries, CPT codes, and strict geospatial boundaries.

Parameters
q Query string (e.g. "knee replacement")
zip Origin US Zip Code (Optional)
radius Search radius in miles (Optional, default: 25)
Shell · Curl
curl "https://healthspend.lol/api/v1/search?q=knee+replacement&zip=90210&radius=20"
GET /API/V1/RATES
Market Context Aggregation

Retrieve the statistical cash-rate spread for a specific procedure within a defined market to instantly identify predatory billing.

Response · JSON
{ "cpt_code": "27447", "market_zip": "90210", "radius_miles": 20, "cash_rate": { "min": 12500.00, "median": 18200.00, "max": 45000.00 } }
POST /API/V1/DISPUTE
Programmatic Enforcement

Generate a legally formatted, deterministic debt dispute letter using a specific hospital's published federal pricing data.

Request Body
{ "hospital_ccn": "050001", "cpt_code": "70551" }
GET /API/V1/COMPLIANCE
Hospital Compliance Matrix

Retrieve the authoritative compliance score and specific violation evidence for any hospital identified by its CMS Certification Number (CCN).

Response · JSON
{ "ccn": "050001", "score": 100, "status": "VERIFIED COMPLIANT" }
